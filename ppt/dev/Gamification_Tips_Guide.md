## 🎯 生活化比喻庫

### M01 典範轉移比喻
```
🍕 AI 協作 = 點外送 vs 自己下廚
- 傳統編程：買菜、洗菜、切菜、炒菜 (累死人)
- AI 協作：點菜、品嚐、給建議 (當美食評論家)

🚗 雙重性原則 = 開車模式
- 週末模式：鄉間小路，慢慢開，欣賞風景
- 工程模式：高速公路，專心開，安全第一

🎭 角色轉換 = 職場升級
- 編碼者：打字員，執行別人的想法
- 架構師：指揮官，設計系統藍圖
```

### M02 工具生態比喻
```
🍳 AI 工具選擇 = 選廚具
- IDE (Cursor)：瑞士軍刀，什麼都能做 80 分
- CLI (Claude Code)：專業主廚刀，一件事做 100 分
- Cloud (v0.dev)：微波爐，快速但功能有限

👑 模型選擇 = 請不同專家
- Reasoning Model：請諸葛亮 (慢但聰明)
- Standard Model：請萬能助手 (平衡高效)
- Budget Model：請實習生 (便宜大碗)

🎮 工具組合 = 遊戲雙開
- 同時開 Cursor + Claude Code
- 一個寫代碼，一個做架構
- 效率翻倍的外掛技巧
```

### M03 上下文工程比喻
```
🏠 上下文工程 = 蓋房子
- CLAUDE.md：房子設計圖 (總體規劃)
- SPEC.md：裝修清單 (具體要求)
- WBS.md：施工進度表 (時間管理)

🐠 AI 記憶對比 = 養寵物
- 對話記憶：金魚 (3 秒就忘)
- 檔案記憶：忠犬 (一次訓練終生記得)

📱 Token 經濟 = 手機流量
- Raw Data：看 4K 影片 (燒錢快)
- Summary：看文字新聞 (便宜有效)
- 原則：先壓縮再餵食
```

### M04 風險控制比喻
```
🚗 AI 風險控制 = 汽車安全系統
- 安全帶：基礎檢查 (必備)
- ABS 煞車：Vibe Check (關鍵時刻)
- 氣囊：錯誤處理 (最後防線)

🏥 風險矩陣 = 醫院檢查
- 綠區：健康檢查 (例行公事)
- 黃區：進一步觀察 (小心監控)
- 紅區：立即手術 (人工介入)

🔒 供應鏈安全 = 食品安全
- 檢查包裝：確認套件名稱
- 看產地：確認維護者身份
- 查保存期限：最近更新時間
```

### M05 實戰演練比喻
```
🏗️ 快速開發 = 樂高積木
- 有標準零件 (React 組件)
- 照說明書組裝 (SPEC.md)
- 一小時做出城堡 (Trello 功能)

🎯 仿製學習 = 臨摹大師作品
- 不用原創構思 (減少認知負載)
- 專注技法練習 (AI 協作手感)
- 逐步掌握精髓 (開發節奏)

🏃 開發節奏 = 跑馬拉松
- 15分鐘：設定環境 (暖身)
- 30分鐘：核心功能 (衝刺)
- 15分鐘：測試部署 (收尾)
```

### M06 未來價值比喻
```
💎 三大價值 = 鑽石切割師
- 洞察力：發現原石 (找到問題)
- 品味：精準切割 (解決方案)
- 擴散力：珠寶展示 (讓人看見)

🏰 職涯護城河 = 古代城防
- 技能多元化：加深河水
- 獨特組合：加高城牆
- 個人品牌：立起旗幟
- 網路效應：結盟其他城主

🌱 微應用生態 = 種植果園
- 從種子開始 (MVP)
- 細心照料 (用戶回饋)
- 逐步擴張 (功能增加)
- 收穫果實 (持續收益)
```
