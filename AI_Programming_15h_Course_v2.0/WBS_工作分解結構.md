# AI 編程速成課程 v2.0 - WBS 工作分解結構

## 📋 專案概述

**專案名稱**：AI 編程速成課程：從情境實戰到企業級整合 v2.0
**專案類型**：情境驅動學習版
**總時長**：15 小時核心課程 + 彈性實戰練習
**完成期限**：2025年3月（預計）
**負責人**：課程開發團隊

---

## 🎯 專案目標

### 主要目標
1. 建立完整的情境驅動學習體系
2. 重新定位 Claude Code 為 AI Agent 交互平台
3. 培養「AI 增強型架構師」能力
4. 建立三層情境題庫系統
5. 整合 Anki 記憶卡長期學習機制

### 成功標準
- [ ] 完成 13 個核心模組內容開發

- [ ] 製作 500+ Anki 記憶卡
- [ ] 完成 10+ 範例專案
- [ ] 建立完整評量體系
