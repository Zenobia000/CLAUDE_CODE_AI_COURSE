# 學習資源 - MCP 與多代理人協作

## 📚 資源概覽

本目錄提供**情境驅動的學習資源**，不是命令參考手冊，而是**問題解決模式庫**。

**設計理念**：
> 不教你「MCP 配置格式是什麼」
> 而是教你「遇到連線失敗時怎麼辦」
